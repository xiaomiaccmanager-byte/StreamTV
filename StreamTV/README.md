# StreamTV — Fake TV Player

Android-приложение, которое превращает любой видеосайт в имитацию телевидения.

## Как работает

1. Вводишь URL сайта (lostfilm.tv, rezka.ag, и др.)
2. Приложение парсит контент и группирует по жанрам
3. Каждый жанр = отдельный телеканал
4. Контент крутится по расписанию как настоящий эфир
5. EPG показывает что идёт сейчас, что было и что будет

## Открыть в Android Studio

1. File → Open → выбрать папку StreamTV
2. Дождаться sync Gradle
3. Run → Run 'app'

## Требования

- Android Studio Hedgehog (2023.1) или новее
- Android SDK 34
- Kotlin 1.9+
- Минимальный Android: 7.0 (API 24)

## Архитектура

```
StreamTV/
├── data/
│   ├── model/       — модели данных (Channel, Show, EpgItem...)
│   ├── parser/      — ContentParser (парсит сайты), StreamExtractor (извлекает видео)
│   └── scheduler/   — EpgScheduler (генерирует расписание)
├── ui/
│   ├── setup/       — экран ввода источника
│   ├── main/        — главный экран (каналы + EPG)
│   └── player/      — плеер на ExoPlayer
└── utils/           — SharedPreferences и утилиты
```

## Поддерживаемые форматы видео

- HLS (.m3u8) — потоковое видео
- DASH (.mpd)
- MP4, MKV — прямые ссылки
- RTMP — потоки

## Известные ограничения

- Парсинг работает только если сайт не блокирует автоматические запросы
- Для сайтов с DRM-защитой потребуется дополнительная реализация
- StreamExtractor извлекает ссылки только с публично доступных страниц
